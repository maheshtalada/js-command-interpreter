Usage Guidelines

      How to use

      • Type Command to excute in the box showing top left panel
      • Traverse Autopopulate list with UP & DOWN arrow keys
      • Press RIGHT arrow or END key to update with autopopulate text
      • Press ENTER + SHIFT key to enter multiline code
      • press ENTER to execute the command / code
